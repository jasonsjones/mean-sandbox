MEAN Sandbox

A playground to explore the MEAN stack

Project Features:

User Registration & Auth

  • utilizes passport-local for user authentication
  • implements Oauth2 registration/ authentication for twitter
  • user's passwords are hashed with a per user salt before writing to db
  • implements data caching to return the list of users to avoid hitting the back-end unnecessarily
  • provides application 'admins' the ability to view (both a list- & grid-view) and modify other users' profiles

TODO Application

  • able to complete a 'todo' item without deleting
  • keeps track of the number of 'todos' remaining
  • simple interface to add a new 'todo'
  • implements data caching for the todo list

ATM Cash Tracker

  • tracks amount spent and amount remaining for atm withdrawals
  • provides a detailed view to add new purchases
  • implements custom directives
  • implements data caching for ATM data
